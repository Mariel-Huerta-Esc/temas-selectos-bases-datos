# Temas Selectos de Bases de Datos

Repositorio del curso **Temas Selectos de Bases de Datos**
(UAM — Trimestre 26P — Grupo CJVS51).

## Objetivo del curso

Este curso aborda el manejo moderno de datos utilizando
bases de datos relacionales, no relacionales y estructuras
basadas en grafos.

A lo largo del trimestre se desarrollarán habilidades para:

- Obtener y transformar datos con Python
- Trabajar con SQL y bases de datos relacionales
- Utilizar modelos NoSQL y documentos JSON
- Analizar datos mediante grafos
- Aplicar análisis estadístico
- Crear visualizaciones y dashboards
- Diseñar pipelines de datos completos
